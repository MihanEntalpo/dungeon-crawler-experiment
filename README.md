# Dungeon Crawler Prototype

HTML+CSS+JS прототип dungeon crawler с процедурной генерацией, базовой боёвкой и системой сохранений через localForage.

## Возможности
- Главное меню: новая игра / загрузка (кнопка загрузки неактивна без сохранения).
- Процедурная генерация подземелья (лабиринт + комнаты + петли).
- Автосохранение карты и сущностей каждые 5 секунд (localForage).
- Туман войны: тайлы открываются в зоне видимости и остаются открытыми навсегда.
- Выход из подземелья: при достижении маркера туман войны отключается полностью.
- Плавное движение персонажа с инерцией и ограничением скорости.
- Поворот персонажа по курсору мыши.
- Ближняя атака по конусу перед персонажем (ЛКМ).
- 3 типа врагов с разными параметрами (скорость, урон, дальность агро, здоровье).
- 100 врагов на карте, базовый ИИ: патрулирование/преследование/атака.
- Простая физика: столкновения со стенами и между мобами.
- Отображение здоровья игрока в виде кругового индикатора и HP врагов полосой.
- Пауза при потере фокуса, кнопка «Продолжить» и переключение по Esc.
- Мобильный режим: два джойстика и атака правым стиком, fullscreen при старте/возобновлении.
- Экран Game Over при нулевом здоровье (перезапуск — обновление страницы).

## Управление
- Движение: стрелки / WASD / ФЦЫВ
- Поворот: мышь
- Атака: ЛКМ
  
Мобильное:
- Левый джойстик — движение
- Правый джойстик — поворот и атака (пока зажат)

## Запуск
Откройте `index.html` в любом современном браузере.

## Файлы
- `index.html` — базовая разметка и подключение скриптов.
- `css/style.css` — стили интерфейса и оверлеев.
- `js/game.js` — точка входа (меню/загрузка/старт).
- `js/classes/` — игровая архитектура (Game, World, EntityManager, Player, Enemy, Map, Fog, Camera, Input, HUD и др.).
- `js/tools.js`, `js/consts.js`, `js/globals.js` — утилиты, константы, глобальные словари ресурсов.
